# Phase-1 Baseline Storage

This folder is the single canonical Phase-1 master input for Phase-2 runs.

Only this `README.md` should be committed. The baseline data files in this
folder are generated locally and are gitignored.

Expected files:
- `Phase1_people_addresses.csv`
- `Phase1_people_addresses.manifest.json`
- `Phase1_people_addresses.quality_report.json`

Do not duplicate multiple large baseline copies for each scenario.
